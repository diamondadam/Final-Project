File Structure
dashboard/
├── server.js   Express — serves public/, single entry point
├── package.json   only dependency: express
└── public/
    ├── index.html   shell: sidebar, alert banner, 4-panel grid, loads scripts
    ├── style.css   charcoal theme, sidebar hover, panel layout
    ├── ws.js   WebSocket connection, reconnect logic, dispatches events
    ├── state.js   shared TwinState store, segment cache, tick history
    ├── charts.js   Chart.js line graph (velocity, accel, speed over time)
    ├── segments.js   Track segment classification panel renderer
    └── orders.js   Work order panel — fetch /work-orders, material select, complete
TO RUN
cd dashboard  &&  npm install  &&  npm start
Requires FastAPI backend already running on port 8000
