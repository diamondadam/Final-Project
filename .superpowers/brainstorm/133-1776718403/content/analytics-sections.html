Analytics Page — Section Contents

Two equal sections. Select any panels you want to remove or swap out.

① Train Telemetry

✓

Speed + Alert Band Chart

Speed line over time with background colored by alert state (green=CLEAR, amber=WARNING, red=DANGER). Expanded version of the Dashboard chart.

✓

Alert State Breakdown

Three stat tiles: ticks spent in CLEAR / WARNING / DANGER, plus % of time at full speed. Gives a quick session summary at a glance.

✓

Train Position Timeline

Step chart showing which segment the train is on each tick — makes it easy to see which segments caused alerts.

② MPC Controller Performance

✓

Commanded vs. Target Speed

Two overlaid lines: v_safe target (what the MPC aimed for based on MAP state) and actual commanded speed (after rate-limiting). Shows the smoothing effect of the QP solver.

✓

Segment Health Heatmap

Grid of segments × ticks, cell colored by MAP state (green/amber/red). Shows which segments were flagged and for how long — the MPC's decision input.

✓

Per-Segment Belief Convergence

Small multiples: one mini chart per segment showing how P(Healthy), P(Degraded), P(Damaged) evolved tick-by-tick. Shows how quickly the Bayesian tracker converged.
